Beacon Tracking and Obstacle Avoidance Robot

Brief Description

The aim of this group project was to develop the hardware and software for an infrared (IR) beacon-tracking, obstacle avoiding mobile robot. The hardware component included designing, building and testing a PIC18f2221 microcontroller based circuit to control the autonomous robot.

Technical Work

I worked in a team of three to design the circuit based around a PIC18f2221 microcontroller which was then assembled on a strip board and installed into the chassis. The microcontroller was programmed using C and rigorously tested using the V-model. The key functions of the robot included orienting itself in space using wheel encoders and IR sensors, maneuvering around obstacles and stopping at the beacon.

The software component involved using C programming to meet the following requirements:
  - Startup sequence with time delays and LED flashing.
  - Scanning room for the beacon by revolving the robot and using an IR receiver.
  - Identifying and proceeding towards the beacon while avoiding low-height obstacles.
  - Sensing and stopping at the beacon.
