AirBnB-Inspired Booking Management System

Brief Description

The aim of this project was to create a booking management system (similar to AirBnB) from scratch for users to host, search for and reserve properties using Java and MySQL.

Technical Work

The backend was designed using MySQL and MySQL Workbench while the frontend app and database connection was programmed using Java. Agile methodology was used to deploy different functionalities efficiently until all requirements were met: 1. Storyboards were created based on user flows (e.g., all steps leading up to a user reserving a property) 2. Templates were designed using Java SWING 3. Functionality was added to the empty templates 4. System was tested and improved iteratively I predominantly worked on steps 1, 3 and 4 where I designed, developed, and tested functionality in Java and ensured the Java-MySQL interface worked smoothly.
