Smart Burglar Alarm

Brief Description

The goal of this project was to create a smart burglar alarm that would unlock a locked door if a user whose face and passcode was verified and trip an alarm in case of a break-in.

Technical Work

The system’s key functionalities included verifying an existing user by face and passcode, and using magnetic lock sensors and PID movement sensors to sense break-ins. I worked on the electronic connectivity, and functionality programming with my teammates on an Arduino UNO microcontroller. The facial recognition was implemented using large datasets of the team’s faces on MATLAB using the Computer Vision System and Machine Learning and Statistics Toolboxes.
